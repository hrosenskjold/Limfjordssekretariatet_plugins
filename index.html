Installation i QGIS

  1. Åbn QGIS
  2. Gå til Plugins → Manage and Install Plugins → Settings
  3. Klik Add under "Plugin Repositories"
  4. Indsæt denne URL: 
    https://Limfjordssekretariatet.github.io/Limfjordssekretariatet_plugins/plugins.xml
  5. Gå til fanen All og søg efter det ønskede plugin
  6. Klik Install Plugin
